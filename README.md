This project has been developed in C++ and is comptatible on Linux and Windows.

## Dependencies

- C++ 17 Compiler
- CMake

## Build and run

### Linux
```bash
git clone https://github.com/AEnguerrand/cpp_zia.git
cd cpp_zia
python build.py
cd build
./cpp_zia
```

### Windows
```bash
git clone https://github.com/AEnguerrand/cpp_zia.git
cd cpp_zia
python build.py
cd build
./cpp_zia.exe
```
## Console

This project has a CLI. To discover all the available commands, type :
